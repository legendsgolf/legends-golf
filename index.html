📖 Tournament Workflow

Complete Tournament Process:

  1. Setup - Admin creates tournament, adds players, configures course
  2. Score Entry - Players enter scores via player interface (auto-validates with partner)
  3. Acceptance - Admin reviews and accepts validated scorecards
  4. View Results - Check individual leaderboards (Stableford, Skins, Eclectic)
  5. Team Setup - Create teams for Best Ball and Ambrose (optional)
  6. Special Prizes - Enter long drive and NTP winners (optional)
  7. Presentation - Open presentation page, select round, announce winners!
  8. Next Round - Repeat for additional rounds

✨ Key Features

✅ Anti-Cheat System

Players validate each other's scores. Admin must accept before locking.

💾 Browser Storage

All data saved in browser. No server needed. Export/import as backup.

📱 Mobile Friendly

Players can enter scores on their phones on the course.

🎯 Live Stableford

See Stableford points instantly as scores are entered.

🏅 Countback

Automatic countback resolution (back 9, 6, 3, last hole).

🖨️ Print Ready

Presentation page prints beautifully for physical handouts.
