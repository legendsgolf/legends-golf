📋 Tournament Day Run Sheet

The whole event in one list. Everything below happens on this laptop, in this browser.

Before the event
  • Tournament Setup — players, handicaps, course, rounds
  • Admin Controls — open Round 1, post any notices
  • Cloud Admin → Publish to Cloud — sends everything to players' phones
  • Send the Player App link (below) to the group
    • During the round
  • Watch scores arrive live in Cloud Admin — you do nothing
  • Flip the Live Leaderboard switch on/off whenever you like
    • After the round
  • Cloud Admin → Import All Matched Scorecards
  • Fix any wrong numbers in the Score Editor
  • Open the results boards, then run the Presentation at dinner
  • Back up: export your data (Tournament Setup page) to a USB or email it to yourself

    • 📱 Links to Send Players

    These are the only two links players ever need.

    Player App https://legendsgolf.github.io/legends-golf/legends-player.html
    Ambrose (captains) https://legendsgolf.github.io/legends-golf/legends-ambrose.html
    Tip for players: open the link once, then "Add to Home Screen" — it becomes an app icon and works with no signal on the course.
